To write and run C programs, you need a **C compiler** like **GCC**.

### Install GCC (Linux & macOS)

```bash
sudo apt update
sudo apt install gcc
```

## Install GCC (Windows)

- Download and install MinGW from mingw-w64.org
- Add MinGW to system PATH
